An integrated, end-to-end desktop application built for the **Super-Lee** supermarket chain. This system transitions the company from manual, paper-based tracking to a fully centralized digital platform, integrating Human Resources, Inventory, Suppliers, and Transportation logistics.

## 🛠️ Tech Stack & Tools
* **Language:** Java 17
* **UI Framework:** JavaFX (Controls & FXML)
* **Database:** SQLite (JDBC) - Local Persistent Storage
* **Build Tool:** Apache Maven
* **Testing:** JUnit 5.10.2
* **Design & Modeling:** [draw.io](https://draw.io) (Used for all UML, Sequence, and Use-Case diagrams located in the `docs` folder)

## 🚀 How to Run the Application
The system is packaged as a standalone executable JAR, requiring no external database installations.
